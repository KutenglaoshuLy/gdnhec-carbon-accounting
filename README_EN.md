# Public Institutions Carbon Accounting System
# 公共机构碳排放核算系统

A web-based carbon accounting tool for public institutions based on JS_T 303-2026 standard.

基于《公共机构碳排放核算指南》（JS_T 303-2026）标准开发的公共机构碳排放核算工具。

## Features / 功能特点

- ✅ Fossil fuel combustion emissions calculation / 化石燃料燃烧排放计算
- ✅ External electricity consumption emissions / 外购电力蕴含排放计算
- ✅ External heat consumption emissions / 外购热力蕴含排放计算
- ✅ Provincial emission factor database / 省级排放因子数据库
- ✅ PDF report export / PDF 报告导出
- ✅ Real-time calculation / 实时计算

## Quick Start / 快速开始

### Use Directly / 直接使用

Just open `index.html` in your browser, or deploy to GitHub Pages / Netlify / Vercel.

直接在浏览器中打开 `index.html`，或部署到 GitHub Pages / Netlify / Vercel。

### Local Development / 本地开发

```bash
# Using Python
python -m http.server 8000

# Visit / 访问 http://localhost:8000
```

## Usage / 使用方法

1. Fill in basic information (institution name, year, province) / 填写基本信息
2. Enter energy consumption data / 输入能源消耗数据
3. Click "Start Calculation" / 点击"开始计算"
4. Export PDF report / 导出 PDF 报告

## Technology Stack / 技术栈

- HTML5
- CSS3 (Gradient design / 渐变设计)
- Vanilla JavaScript (No frameworks / 无框架依赖)

## Standard Basis / 标准依据

- 《公共机构碳排放核算指南》（JS_T 303-2026）
- GB/T 31342 公共机构能源审计技术导则
