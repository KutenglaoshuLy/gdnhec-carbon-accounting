# 公共机构碳排放核算系统

## 📋 项目简介

本系统依据《公共机构碳排放核算指南》（JS_T 303-2026）标准开发，为公共机构提供便捷、准确的碳排放量计算工具。

## 🎯 功能特点

### 1. 完整的核算范围覆盖
- ✅ **化石燃料燃烧排放**：煤炭、天然气、汽油、柴油等 10 种燃料
- ✅ **外购电力蕴含排放**：支持省级电网因子自动匹配
- ✅ **外购热力蕴含排放**：支持自定义排放因子

### 2. 标准化计算方法
严格遵循标准第 8 章规定的核算方法：
- 排放总量 = 直接排放 + 间接排放
- 直接排放 = ∑(燃料消耗量 × 排放因子)
- 间接排放 = 电力排放 + 热力排放

### 3. 智能化数据处理
- 🔄 实时计算：输入数据即时显示分项排放量
- 📊 结果可视化：卡片式展示四大核算结果
- 📝 报告导出：一键生成 Markdown 格式核算报告

### 4. 权威参数数据库
内置标准附录 A 全部缺省值：
- 化石燃料排放因子（表 A.1）
- 省级电力平均二氧化碳排放因子（表 A.2）
- 热力排放因子默认值 0.11 tCO₂/GJ

## 📖 使用步骤

### 第一步：填写基本信息
1. **机构名称**：填写单位全称
2. **核算年度**：选择统计周期年份
3. **所在省份**：选择所在地（用于匹配电力排放因子）

### 第二步：填报能源消耗数据

#### 1. 化石燃料燃烧排放
根据实际消耗的燃料品种填写：
- **无烟煤/烟煤**：单位：吨 (t)
- **燃料油/汽油/柴油/一般煤油**：单位：升 (L)
- **天然气/焦炉煤气/管道煤气**：单位：立方米 (m³)
- **液化石油气**：单位：千克 (kg)

> 💡 提示：无法确定煤种时，按保守原则以无烟煤填报

#### 2. 外购电力蕴含排放
- **外购电量**：填写总购入电量（MWh）
- **其中绿电**：如有绿电直连，填写具体数值（排放因子按 0 计）
- **转供电量**：向居民、商户转供的电量（如有）

> 💡 数据来源建议：
> - 优先采用电能计量装置记录数据
> - 也可采用电费发票、结算单中的电量数据

#### 3. 外购热力蕴含排放
- **外购热量**：填写实际购入热量（GJ）
- **转供热量**：对外转供的热量（如有）
- **排放因子**：默认 0.11 tCO₂/GJ，可根据最新数据调整

> 💡 跨年度数据处理：
> 对于按供热季统计的数据，应按供热季对应的自然年度进行合理拆分

### 第三步：计算并查看结果
1. 点击 **"🔢 开始计算"** 按钮
2. 查看碳排放总量及分项排放
3. 查看详细计算过程和公式

### 第四步：导出报告
点击 **"📄 导出报告"** 按钮，下载 Markdown 格式的核算报告

## 📐 计算公式详解

### 1. 排放总量计算（公式 1）
```
E总 = E直接 + E间接
```

### 2. 直接排放计算（公式 2）
```
E直接 = ∑(FCi × EF i,直接)
```
其中：
- FCi：第 i 种化石燃料消耗量
- EF i,直接：第 i 种化石燃料排放因子

### 3. 间接排放计算（公式 3-5）
```
E间接 = E电力 + E热力

E电力 = ∑(ADj,外购电力 × EFj,外购电力) - ∑(ADk,转供电力 × EFk,转供电力)

E热力 = AD外购热力 × EF外购热力 - AD转供热力 × EF转供热力
```

## ⚠️ 注意事项

### 1. 核算边界
- 覆盖运行边界内固定设施和移动设施产生的碳排放
- 涉及家属区、周转住房、对外出租部分的，应与《公共机构能源资源消费统计调查制度》保持一致

### 2. 不纳入核算范围的情况
- 向居民、商户等转供电对应的碳排放
- 向居民、商户等转供热对应的碳排放
- 分布式光伏发电系统自发自用电力

### 3. 数据一致性
- 集中供热等按供热季统计的能源消费数据，应按照供热季对应的自然年度进行合理拆分
- 确保与电力、燃料等其他能源消费数据在时间尺度上的一致性

### 4. 数据质量要求
- 定期维护能源资源计量器具
- 留存数据来源、拆分核算、参数选取等过程记录
- 确保数据可追溯、可核查
